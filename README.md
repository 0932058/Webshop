# Webshop

# How to watch files
- Go to the main directory: ReactWebShop
- Start CMD or powershell in that directory
- Type the command: ./node_modules/.bin/webpack -w

# To Code
- Go to the main directory: ReactWebShop
- Start CMD or powershell in that directory
- Type the command: code. 

# To run the program
- Go to the main directory: ReactWebShop
- Start CMD or powershell in that directory
- Type the command: dotnet start
